Whats for dinner?
My app will give the user access to an assortment of restaurants, with show pages for each restaurant. Also there will be an option to click a button that will generate a random restaurant to try.

##Technologies Used
-HTML5
-CSS
-Javascript
-JQuery
-Nodejs
-EJS
-MongoDB
